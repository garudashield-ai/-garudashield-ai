### 🧠 Mesin AI "Ultimate Judge"
Alih-alih hanya mengandalkan skor kaku dari sistem heuristik, GarudaShield AI menggunakan arsitektur **AI-First** yang ditenagai oleh **ND-Labs GPT-5**. Sistem akan mengekstrak bukti forensik (dari Nmap, ZAP, Apktool, Tor Proxies) dan menyerahkannya kepada AI untuk mengambil keputusan akhir yang cerdas, sehingga meminimalkan *false-positive*.

### 🕸️ Pemindai Web (Web Scanner)
Mesin intelijen komprehensif untuk tautan (URL):
- 🌐 **Pemetaan Infrastruktur**: Pemindaian *Subdomain* DNS & pelacakan port dengan Nmap.
- 🛡️ **Pemindai Kerentanan**: Terintegrasi dengan OWASP ZAP untuk mencari celah keamanan.
- 🦠 **Deteksi Malware**: Membedah HTML untuk mendeteksi injeksi *JavaScript*, penambang kripto, dan skrip *phishing*.

### 📱 Pemindai APK (APK Scanner)
Analisis mendalam untuk aplikasi Android (APK):
- 🛠️ **Dekompilasi**: Membongkar paksa aplikasi menggunakan `apktool` dan `aapt`.
- 🕵️ **Pengecekan Steganografi**: Menggunakan `exiftool` dan `binwalk` untuk melacak *malware* yang menyembunyikan dirinya di dalam gambar.
- 🔎 **Analisis Heuristik**: Mendeteksi permintaan izin berbahaya (seperti akses baca SMS untuk mencuri OTP) serta melacak *C2 server* atau nomor WhatsApp pelaku.

### Prasyarat Penting
Untuk menjalankan mesin AI "Ultimate Judge", Anda diwajibkan memiliki *API Key* dari ND-Labs:
- 🔑 **Ambil API Key di sini**: [https://app.nd-labs.dev/](https://app.nd-labs.dev/)
- 📖 **Kunjungi Blog Kami**: [https://nd-labs.id/](https://nd-labs.id/)

Masukkan *key* tersebut ke dalam file `.env.local` Anda:
```env
ND_LABS_API_KEY=api_key_anda_disini
```

### 1. Kloning Repositori & Masuk ke Folder
```bash
git clone https://github.com/garudashield-ai/-garudashield-ai.git
cd -garudashield-ai
```


### 2. Auto-Install Dependensi
Kami telah menyediakan *script* instalasi otomatis (Auto-Installer) untuk mempermudah penyiapan sistem (Nmap, ZAP, Apktool, Tor, Node.js). 

**🖥️ Pengguna Linux / Ubuntu (Direkomendasikan)**
Jalankan perintah Bash berikut di terminal Anda:
```bash
sudo bash install.sh
```

**🪟 Pengguna Windows (PowerShell)**
Buka PowerShell sebagai *Administrator* dan jalankan:
```powershell
Set-ExecutionPolicy Bypass -Scope Process -Force; .\install.ps1
```
*(Catatan untuk Windows: Sangat direkomendasikan menggunakan WSL / Ubuntu CLI agar modul pembedah APK dan ZAP berjalan optimal)*.

### 3. Instal Dependensi Node & Jalankan
```bash
npm install
npm run dev
```
Buka browser dan akses (Lokal): `http://localhost:3000`
Atau akses versi Live Demo di: **[garuda.nd-labs.dev](https://garuda.nd-labs.dev)**
